## Short Introduction ##

MxUpdate is a deployment tool  for the Enovia PLM platform to handle administration objects as single update files ([configuration items](CI.md)).

MxUpdate itself is a set of two tools:
  * MxUpdate Update Deployment Tool ([Installation](UpdateInstallation.md), [Usage](UpdateUsage.md), [Maven Site](http://update.mxupdate.org))
  * MxUpdate Eclipse Plug-In ([Installation](EclipseInstallation.md), [Usage](EclipseUsage.md), [Maven Site](http://mxupdate.googlecode.com/svn/site/mxupdate-eclipse/index.html))

For more information see the Wiki [start](Start.md) page.

The Wiki pages itself are concatenated to one PDF manual. The manual could be [downloaded](http://code.google.com/p/mxupdate/downloads/list).
